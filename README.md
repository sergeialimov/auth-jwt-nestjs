# nestjs-jwt-auth

## Description
Authorization by JWT in Nest.js app


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# build
$ yarn build

# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
